## Cliproxy 购买配置流程
### 长效静态IP 购买配置流程
1. 选择国家/地区，支持 7 天、30 天、90 天等有效期，提供家庭原生住宅 IP 和家庭广播 IP 两种类型。
2. 完成购买后进入 Dashboard，在对应套餐下选择**账密模式**。
3. 在静态余额板块点击**提取**，获取 IP 信息。
4. 提取后在【使用记录】中查看 IP：端口、用户名、密码等信息，可复制到 DuoPlus 配置。
5. 支持扫码快速粘贴到手机使用。
6. 点击【使用】可选择账密/非账密认证模式，生成对应格式命令。
7. 点击【续费】可续期 7/30/90 天。
8. 点击【编辑】可修改账号密码及 IP 备注。

### 动态住宅代理 IP 配置流程
1. 购买动态住宅代理（标准/企业版），企业版支持生成分发 CDK。
2. 特点：流量**永久有效**，无过期限制。
3. 购买后进入**流量代理-账密模式**，生成账密命令。
4. 选择国家、州省、城市、会话类型：
   - **Sticky IP（粘性 IP）**：规定时间内 IP 不变，最长支持 120 分钟会话
   - **Rotating IP（轮转 IP）**：每次请求自动更换新 IP
5. 选择转发机器（主机名:端口），建议选择距离目标站点或本地较近的节点。
6. 可使用测试命令在命令提示符中验证代理可用性。
7. 复制主机:端口、账号、密码，配置到云手机或支持 HTTP/SOCKS5 的程序中。

## Hubstudio 中设置 Cliproxy 步骤
1. **完成 Hubstudio 下载与安装**，点击【新建环境】。
2. **填写代理信息**
   - 代理协议选择 **SOCKS5**
   - 粘贴 Cliproxy 控制台生成的：代理主机、代理端口、代理账号、代理密码
3. **检测代理**
   - 点击【检查代理】
   - 显示绿色即为连通正常
4. **完成设置并访问**
   - 点击【完成】，返回环境界面
   - 点击【打开】，即可使用该代理 IP 访问网页
